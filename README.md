## Compares two configuration files and shows a difference.
### Supports
   json, yaml, ini

**Installation**
```
$ npm i diff-ivstyopa
```
**Usage**
```
gendiff [options] <firstConfig> <secondConfig>
```
**Options**
```
Options:

  -V, --version        output the version number
  -f, --format [type]  output format
  -h, --help           output usage information
```
